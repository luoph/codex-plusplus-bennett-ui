# Bennett's UI Improvements

Codex++ tweak with small quality-of-life UI improvements for the Codex desktop app.

## Features

- Hide upgrade prompts
- Show usage in the sidebar
- Show message metrics on assistant messages
- Square/match sidebar styling
- Compact sidebar action grid
- Project row backgrounds and color theming

## Install

Drop this folder into:

```sh
~/Library/Application Support/codex-plusplus/tweaks/
```

Then reload tweaks from Codex++.

## Manifest

Tweak id: `co.bennett.ui-improvements`
